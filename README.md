CYBERVERSE is a Flask-based cybersecurity operations platform that combines multiple security tools into a single dashboard.

The platform helps users analyze passwords, detect phishing URLs, perform basic vulnerability assessments, analyze digital evidence, review threat intelligence, and generate incident response recommendations.

## Features

### Password Security Analyzer
- Password strength calculation
- Security score generation
- Crack time estimation

### Phishing URL Detector
- Suspicious keyword detection
- HTTPS validation
- Risk scoring

### Vulnerability Scanner
- Software vulnerability lookup
- CVE identification
- Severity assessment

### Digital Forensics Analyzer
- MD5 hash generation
- SHA1 hash generation
- SHA256 hash generation

### Threat Intelligence Feed
- IP reputation analysis
- Risk level classification
- Threat categorization

### AI Incident Response Assistant
- Incident severity analysis
- Automated response recommendations
